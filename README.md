# LX Language (Version 1.0)

LX is a custom-built, statically-typed programming language designed for clarity, modularity, and control. It runs on a custom Virtual Machine (VM) and compiles source code into optimized bytecode.

## Key Features
* **Hybrid Execution:** High-level structured loops (`loop_from`) combined with low-level control flow (`jump`, `label_mark`).
* **Strongly Typed:** Explicit type declarations for `short`, `long`, `float`, and more.
* **Modular:** Python-style imports with `with_get` and support for module aliasing.
* **Readable Syntax:** Human-centric keywords like `VERUM`, `FALSUM`, and `scream:`.
